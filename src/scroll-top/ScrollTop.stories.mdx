# Back to top `<scroll-top>`

The goal of this component is to provide an unopinionated component to
scroll back to the top of the page.

## Usage

```html
<scroll-top hidden>
  <button>Back to top</button>
</scroll-top>
```

## Demo

export const Template = ({ label, ...args }) => {
  try {
    customElements.define('scroll-top', ScrollTop)
  } catch (e) {
    // do nothing this is custom elements
  }
  return `

    <scroll-top class="scrolltop" hidden>v</scroll-top>

};
