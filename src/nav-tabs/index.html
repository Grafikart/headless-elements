Style

Here is some help to get you started with styling this element

/* CSS Animation */
[role="tabpanel"] {
  overflow: hidden;
  transition: .3s;
}
[role="tabpanel"][hidden="hidden"] {
  display: block;
  position: absolute;
  top: 0;
  left: 0;
  right: 0;
  pointer-events: none;
  transform: translateY(10px);
  opacity: 0;
}
