ScrollTop

The goal of this component is to provide an unopinionated component to scroll back to the top of the page.

Scroll this page !

Style

Here is some help to get you started with styling this element

/* CSS Animation */
scroll-top {
  position: fixed;
  bottom: 20px;
  right: 20px;
}
html {
  scroll-behavior: smooth;
}
@media (prefers-reduced-motion: reduce) {
  html {
    animation: auto;
  }
}
