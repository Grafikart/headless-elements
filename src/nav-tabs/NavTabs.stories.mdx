# nav-tabs

The goal of this component is to provide an unopinionated component to
create tabs.

### With links

You can use regular links (with anchors) to control what element to display

```html
<nav-tabs class="nav-pills">
  <a href="#tab1">Tab 1</a>
  <a href="#tab2">Tab 2</a>
  <a href="#tab3" aria-selected="true">Tab 3</a>
</nav-tabs>
```

    // do nothing this is custom elements

<button aria-controls="tab3${suffix}" id="btntab3">Tab 3</button>
    ` :
    `
<a href="#tab1${suffix}">Tab 1</a>
<a href="#tab2${suffix}">Tab 2</a>
<a href="#tab3${suffix}" aria-selected="true">Tab 3</a>
    `
}
</nav-tabs>
<div id="tab1${suffix}" hidden="hidden">
  <p>
    Lorem ipsum dolor <a href="https://github.com">github link</a> amet
    consectetur adipisicing elit. Corporis, quis incidunt culpa vel aut
    repellendus pariatur at, sapiente, saepe mollitia magnam! Obcaecati
    eum quasi velit architecto. Cupiditate esse perspiciatis veniam.
  </p>
</div>
<div id="tab2${suffix}" hidden="hidden">
  <p>
    Eum libero nemo excepturi labore exercitationem et? Et, perspiciatis
    maiores ipsum nisi rerum expedita quos amet optio corrupti illum
    exercitationem quasi numquam quae autem esse accusamus recusandae sint
    nulla magnam!
  </p>
</div>
<div id="tab3${suffix}">
  <p>

  </p>
  <p>I'm tab 3 content</p>
</div>
`
};

### With buttons

You can also use buttons (if you don't want anchors navigation).

```html
<nav-tabs class="nav-pills">
  <button aria-controls="tab1" id="btntab1">Tab 1</button>
  <button aria-controls="tab2" id="btntab2" aria-selected="true">Tab 2</button>
  <button aria-controls="tab3" id="btntab3">Tab 3</button>
</nav-tabs>
```
